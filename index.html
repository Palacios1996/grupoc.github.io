Portafolio del Grupo C: Desarrollo Ágil y Uso de Inteligencia Artificial Generativa

Metodología Ágil Utilizada

Implementamos Kanban siguiendo estos principios:

  • Tablero visual con columnas: Pendiente, En Progreso, Revisión y Completado
  • Límite de Trabajo en Progreso (WIP) de 2 tareas por persona
  • Reuniones diarias de sincronización de 15 minutos
  • Priorización de tareas con el método MoSCoW
  • Flujo continuo con entrega incremental

Uso de Inteligencia Artificial Generativa

Durante el desarrollo implementamos:

  • Generación de código con GitHub Copilot
  • Optimización de CSS usando ChatGPT-4
  • Validación de arquitectura con Claude AI
  • Generación de documentación con Gemini
