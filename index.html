Objetivo del Portafolio

El objetivo de este portafolio es demostrar las habilidades adquiridas en el desarrollo ágil de software y la integración de inteligencia artificial generativa en la creación de soluciones efectivas.

Introducción

Bienvenido/a a nuestro portafolio, un espacio donde convergen el desarrollo ágil de software y la inteligencia artificial generativa para crear soluciones innovadoras y eficientes.

  • Desarrollo ágil con enfoque en MVP.
  • Uso de IA generativa como ChatGPT y DALL·E.
  • Código limpio y arquitectura escalable.

Descripción del Problema

La necesidad de un portafolio digital profesional es fundamental. Por ello, desarrollamos una plataforma para mostrar proyectos individuales y fomentar la colaboración.

Metodología Ágil Utilizada

Aplicamos Kanban para organizar tareas, flujos de trabajo y responsabilidades. Usamos herramientas como Trello y reuniones semanales para avanzar iterativamente.

Uso de Inteligencia Artificial Generativa

Utilizamos IA generativa en distintas áreas:

  • ChatGPT para generar textos explicativos y documentación técnica.
  • DALL·E para crear imágenes conceptuales para el diseño de interfaces.
  • Refactorización de código usando sugerencias de IA para mantener buenas prácticas.
